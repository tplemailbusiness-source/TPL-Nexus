# TPL Nexus

## Vision

TPL Nexus is a private music streaming platform built for my family.

The goal is not to clone Apple Music, but to create a premium streaming experience inspired by Apple's design philosophy while having its own unique identity.

TPL Nexus should feel elegant, cinematic, modern, smooth, and immersive.

The application should work beautifully on desktop, tablet, and mobile devices.

Eventually it should become an installable Progressive Web App (PWA).

---

# Design Philosophy

Think:

* Apple Music
* macOS
* Dynamic Island
* Disney+
* Glassmorphism
* Cinematic UI

Not:

* Spotify clone
* Generic Bootstrap website
* Material Design app

The interface should have its own personality.

---

# Visual Identity

Primary colors

* Dark Navy
* Deep Blue
* Cyan
* Teal

The background should slowly animate with liquid gradients.

Album artwork should influence the colors of the interface.

Transitions should feel smooth and natural.

---

# Design Rules

* Large album artwork
* Large spacing
* Rounded corners
* Frosted glass panels
* Soft shadows
* Subtle animations
* Smooth page transitions
* Minimal clutter
* Mobile-first design

Nothing should feel cramped.

Everything should breathe.

---

# Platform Goals

The same codebase should support:

* Desktop
* Laptop
* Tablet
* Mobile

The layout should adapt automatically.

Do NOT create separate desktop and mobile websites.

Write clean code.

Separate responsibilities.

Keep files organized.
